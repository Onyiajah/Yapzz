# YapzZ — Marketing Site (UI Kit)

A click-thru recreation of the YapzZ marketing site — single long-form page with hero, capability sections, a compliance-receipt explainer, and footer. Style matches the design system: paper background, signal-orange accents, monospace serials, no gradients, no glass.

## Files
- `index.html` — loads all components and mounts the page
- `Sections.jsx` — `Header`, `Hero`, `Capabilities`, `ReceiptExplainer`, `Sectors`, `CTA`, `Footer`

## Source of truth
No marketing site or codebase was supplied — this is derived from the brand brief and the design system in this repo.
